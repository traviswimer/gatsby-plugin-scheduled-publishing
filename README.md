> `gatsby-plugin-mdx-scheduled-publishing` helps you auto-publish MDX pages without needing a CMS.

## Getting Started

### Install

**Yarn:**

```shell
yarn add gatsby-plugin-mdx-scheduled-publishing
```

**NPM:**

```shell
npm install gatsby-plugin-mdx-scheduled-publishing
```
